# 🏥 AI Disease Predictor – Smart Health Diagnosis System

An intelligent Machine Learning-based **Disease Prediction System** that analyzes user-provided symptoms and predicts possible diseases with high accuracy.

This project is designed to assist users with **early health insights** by combining data-driven models and intelligent automation.

---

## 🚀 Features

### 🧠 Disease Prediction Engine
- Predicts diseases based on symptoms  
- Uses ML algorithms like:
  - Support Vector Machine (SVM)  
  - K-Nearest Neighbors (KNN)  
  - Random Forest  
- Provides accurate and fast predictions  

---

### ⚡ Automated Model Selection
- Compares multiple models  
- Selects the best-performing model automatically  
- Improves prediction reliability  

---

### 📊 Data Preprocessing
- Handles missing values and noisy data  
- Encodes categorical features  
- Normalizes input data  

---

### 📈 Performance Evaluation
- Accuracy, Precision, Recall, F1-score  
- Model comparison for better decision-making  

---

### 🌐 User-Friendly Interface
- Simple UI to input symptoms  
- Instant prediction results  
- Easy to use for non-technical users  

---

## 🛠️ Tech Stack

- Programming Language: Python  
- Machine Learning: Scikit-learn  
- Backend: Flask / FastAPI  
- Frontend: HTML, CSS, JavaScript  
- Libraries: NumPy, Pandas, Matplotlib  

---

## ⚙️ How It Works

1. User inputs symptoms  
2. Data is preprocessed  
3. Features are prepared  
4. Best ML model is selected  
5. Disease is predicted and displayed
